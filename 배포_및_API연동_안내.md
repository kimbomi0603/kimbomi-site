## 2. GitHub Pages로 웹 연결하기

> 계정 생성·로그인은 보안상 직접 진행하셔야 합니다. 아래 순서만 따라 하시면 됩니다.

### ① 저장소 만들기
1. <https://github.com> 로그인 → 우측 상단 **＋ → New repository**
2. Repository name 입력(예: `gwangju-parkjeongha`) → **Public** 선택 → Create

### ② 파일 업로드
1. 저장소 화면에서 **Add file → Upload files**
2. 이 폴더(`박정하홍보페이지`) **안의 모든 파일·폴더**를 끌어다 놓기
   - `index.html`, `assets/` 폴더, `CNAME`, `.nojekyll` 까지 **폴더 구조 그대로** 올립니다.
3. **Commit changes**

### ③ Pages 켜기
1. 저장소 **Settings → Pages**
2. Source: **Deploy from a branch** → Branch: **main** / 폴더: **/(root)** → Save
3. 1~2분 후 `https://<사용자명>.github.io/<저장소명>/` 으로 사이트가 열립니다.

### ④ 도메인(광주박정하.kr) 연결
저장소에는 이미 `CNAME` 파일(`xn--hc0bu8zl5im0ac66b.kr` = 광주박정하.kr의 퓨니코드)이 포함돼 있습니다.

**GitHub 쪽**
- Settings → Pages → **Custom domain** 에 `광주박정하.kr` 입력 → Save
- 인증서 발급 후 **Enforce HTTPS** 체크

**도메인 등록업체(가비아/후이즈 등) DNS 설정** — `광주박정하.kr`은 최상위 도메인이므로 **A 레코드**를 사용합니다.

| 타입 | 호스트 | 값 |
|---|---|---|
| A | @ | 185.199.108.153 |
| A | @ | 185.199.109.153 |
| A | @ | 185.199.110.153 |
| A | @ | 185.199.111.153 |
| CNAME | www | `<사용자명>.github.io` |

> DNS 반영에는 보통 10분~수 시간이 걸립니다. 반영 후 `https://광주박정하.kr` 로 접속됩니다.

## 3. 지방재정365 API 실시간 연동 (인증키 입력 시 자동 갱신)

기본 상태에서도 사이트는 **광주 북구 2026년 본예산 확정 수치(지방재정365 공시 기준)**를 정확히 보여줍니다.
아래 설정을 하면 공공데이터포털 API로 **실시간 갱신**되도록 전환됩니다.

### ① 인증키 발급
1. **공공데이터포털** <https://www.data.go.kr> 회원가입·로그인
2. **‘행정안전부 지방재정365’** 검색 → 오픈API 활용신청
   - 참고: <https://www.data.go.kr/data/15138709/openapi.do>
3. 승인 후 **마이페이지 → 인증키 발급현황**에서 **일반 인증키(서비스키)** 복사

### ② 키 입력 (`assets/data.js` 맨 아래 `LOFIN` 부분)
```js
const LOFIN = {
  enabled: true,                 // ← false 에서 true 로 변경
  serviceKey: "여기에_발급받은_인증키_붙여넣기",
  proxy: "",                     // (CORS 차단 시 아래 ③ 참고)
  region: { sido: "광주광역시", sigungu: "북구" },
  year: 2026
};
```
저장 후 GitHub에 다시 업로드하면 끝입니다. 키가 정상 동작하면 예산 페이지 상단에 **“● 실시간 연동”** 배지가 표시되고, KPI·차트가 API 값으로 자동 갱신됩니다.

### ③ (중요) CORS 차단 시 — 프록시 또는 서버리스
공공데이터포털 API는 브라우저에서 직접 호출하면 **CORS 정책으로 차단**되는 경우가 많습니다. 이때는 둘 중 하나를 사용하세요.

- **간편(테스트용):** 프록시 서비스의 주소 접두어를 `proxy`에 입력
  ```js
  proxy: "https://corsproxy.io/?"   // 예시. 운영용으로는 권장하지 않음
  ```
- **권장(운영용):** Cloudflare Workers / Vercel 등에 **인증키를 숨긴 서버리스 함수**를 만들어 그 주소를 `proxy`에 넣기
  → 인증키가 외부에 노출되지 않아 안전합니다. (공개 저장소에 키를 그대로 올리면 노출 위험)

### ④ 응답 필드 매핑
API 응답의 실제 필드명은 발급받은 문서에 따라 다를 수 있습니다.
`assets/budget-api.js`의 `parse()` 함수 안 필드명(`budgetAmt`, `generalAmt`, `specialAmt` 등)을 문서에 맞춰 수정하면 됩니다. 매핑이 맞지 않으면 자동으로 **기존 공식 확정 수치**로 표시됩니다(사이트는 항상 정상 작동).

## 5. 콘텐츠 수정 방법
대부분의 내용은 **`assets/data.js`** 한 곳에서 바꿀 수 있습니다.
- 연락처·슬로건·선거사무소: `CAND`
- 예산 수치·분야별 사업: `BUDGET`
- 인구 지표: `DEMO`
- 동별 공약: `PLEDGES`
- 약력: `BIO`

민원·정책 제안 메일 수신처는 `CAND.email`(현재 `didvkwls@hanmail.net`)로 연결돼 있습니다.
