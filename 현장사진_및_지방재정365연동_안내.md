# 현장사진 + 지방재정365 실시간 연동 안내

## 바뀐 점
- 현장 속으로 섹션 신설(홈+활동·공보) — 현장사진이 옆으로 흐르는 슬라이드
- 반복되던 공보물 이미지 제거(공보물 PDF·면별 보기는 유지)
- 페이스북·유튜브 버튼 연결
- 후보 소개 사진 크기 단정하게 고정(최대 300px)
- 지방재정365 실시간 연동 코드 적용(Cloudflare Worker 연결 시 작동)
- 사진 지연로딩으로 모바일 속도 개선

## 1. 현장사진 교체/추가
- 사진은 assets/img/ 에 field-01.jpg ~ field-10.jpg 로 들어있습니다.
- 설명 문구는 assets/data.js 의 FIELD 목록에서 수정할 수 있습니다.

## 2. 지방재정365 실시간 연동 (Cloudflare Worker, 선택)
정적 사이트는 CORS 때문에 API 를 브라우저에서 직접 못 부릅니다.
무료 Cloudflare Worker 로 인증키를 숨겨 호출합니다.
(예산은 연 1회 갱신이라 선택사항 — 안 해도 실제 확정 수치가 표시됩니다.)

1) https://dash.cloudflare.com 가입(무료) 후 로그인
2) Workers & Pages → Create → Create Worker → 이름 예: parkjeongha-budget → Deploy
3) Edit code 에서 cloudflare-worker.js 내용 전체 붙여넣고 Deploy
4) 그 Worker → Settings → Variables and Secrets → Add
   - Type: Secret / Name: LOFIN_KEY / Value: (보내주신 지방재정365 인증키) → Save
5) Worker 주소 복사 (예: https://parkjeongha-budget.이름.workers.dev)
6) 확인: 브라우저에서 https://(그주소)/?fyr=2026&laf=광주북구 → JSON 이 보이면 성공
7) assets/data.js 의 LOFIN.proxy 에 그 주소 붙여넣고 깃허브 업로드
